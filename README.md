# 📘 Sistema de Enseñanza de Programación Básica (SEPB)

![Versión](https://img.shields.io/badge/version-1.0.0-blue)
![Licencia](https://img.shields.io/badge/license-MIT-green)

Bienvenido al **Sistema de Enseñanza de Programación Básica**. Esta plataforma ha sido desarrollada con el objetivo de facilitar el aprendizaje de la lógica de programación para principiantes, ofreciendo una ruta estructurada desde los conceptos más simples hasta estructuras de datos básicas.

---

## 🚀 Características Principales

* **Ruta de Aprendizaje Interactiva:** Módulos que cubren Variables, Condicionales, Ciclos y Funciones.
* **Consola de Práctica:** Espacio integrado para ejecutar y probar fragmentos de código.
* **Evaluación Continua:** Quizzes al finalizar cada módulo para validar conocimientos.
* **Gestión de Usuarios:** Registro de progreso para que el estudiante retome donde dejó.
* **Interfaz Simple:** Diseñada para evitar la sobrecarga cognitiva en nuevos estudiantes.
